INTRO

I recently got a Raspberry Pi 3B to connect to my CRT for watching stuff and emulating games. There are tutorials, but nothing that really encapsulated exactly what I was looking for so I'm making my own tutorial as I work on it. Maybe this help one person in the world or at least be an interesting journey to follow idk


i will probably continue to update this page as i do/learn more

MY GOAL

I'm aiming to make this Pi into a (mostly) all-in-one system for my CRT. I mainly want to:


> Watch movies/shows/anime via my plex server

> Emulate some retro games, primarily SNES & GBA.

> Watch youtube

> Stream PC Games to my CRT

> Web browsing

WAT U NEED

> Raspberry Pi

I'm using a 3B, but any pi 3 works. The Pi 4 has issues with composite output so dont get that

> Case/Power Supply

use an official Pi power supply or else. case isnt strictly necessary but just at least get a cheap one to be safe

> Aux to Composite Cable


> Micro SD Card

Get one from a reputable brand, some people have had issues with cheaper ones.

> Keyboard & Controller

Controller is optional, feels a whole lot better though.

RETROPIE SETUP

Ok lets get started!!! First off, I downloaded the operating system for the pi onto the Micro SD Card. you can find the OS Imager here. I chose Retropie. Just use the program to image your Micro SD card then there will be 2 partitions. Windows wont be able to read one of them, that is normal.

In the one that windows can read, open "config.txt". In this, youre gonna paste this at the bottom:

sdtv_mode=0
sdtv_aspect=1

"sdtv_mode=0" makes the output NTSC 480i. you can set emulators to be 240p later. the "sdtv_aspect=1" just gives a 4:3 output.


Now you are ready to plug into your CRT!!! once it loads up and you set up your controller, you should probably set up wifi & SSH, it makes stuff easier. to do this, go to:

WIFI: RASPI-CONFIG > System Options > Wireless LAN
SSH: RASPI-CONFIG > Interface Options > SSH

with ssh you can remotely connect to the pi and do stuff from your pc/laptop. i used it a lot to adjust overscan so the image fits properly on my crt. you can also use it to transfer files like roms and whatnot. to connect to the pi using SSH, you need to get your pi's IP. theres a show ip option in the main menu. then open powershell on windows or terminal on mac and put this:

ssh pi@[Pi's IP]

the default password is "raspberry". once you get in you should change your password by typing "passwd".
